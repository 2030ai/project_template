# 2030AI Project Template

Шаблон проекта для работы с Claude AI и Cursor

## Структура документации

- `claude.md` — правила работы Claude над проектом
- `ARCHITECTURE.md` — описание архитектуры (создается при необходимости)
- `ADR.md` — архитектурные решения (создается при необходимости)
- `development-history.md` — история разработки (создается при необходимости)
- `.claude-context.md` — контекст между сессиями (создается по необходимости)

## Быстрый старт

1. Клонируйте репозиторий
2. Изучите `claude.md` для понимания правил работы
3. Начните разработку

## Документация

Подробные правила работы и стандарты проекта описаны в `claude.md`.
